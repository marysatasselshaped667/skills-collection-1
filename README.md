## 📁 File and folder layout

A large skill collection often uses a structure like this:

- `skills/` for reusable skill folders
- `prompts/` for prompt templates
- `workflows/` for step-by-step task flows
- `references/` for notes and source material
- `scripts/` for helper files
- `assets/` for images, icons, and support files

This layout helps you move from one task to the next without searching through loose files.

## 🔎 Tips for better results

- Keep the release folder in a place you can find later
